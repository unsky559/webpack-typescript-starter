# Webpack Typesctipt starter

## What`s inside?
- eslint-plugin
- eslint (airbnb)
- prettier
- css-loader
- sass
  - sass-loader
- style-loader
- typescript
- webpack
  - html-webpack-plugin
  - copy-webpack-plugin

## Commands

For start dev server use
```
> npm run serve
```
For build use
```
> npm run build
```
Before deploy use
```
> npm run build:prod
```
For start lint tests use
```
> npm run lint
```
